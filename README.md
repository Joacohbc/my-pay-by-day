<h1 align="center">My Pay By Day</h1>

<p align="center">
  Una aplicación web full-stack para gestionar y realizar un seguimiento de tus registros de pago diarios. Toma el control de tus finanzas personales con una interfaz rápida y una arquitectura moderna, inteligente y robusta.
</p>
